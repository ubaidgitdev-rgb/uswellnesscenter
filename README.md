## ğŸš€ Getting Started

### Prerequisites

-   Node.js (v18+)
-   MongoDB (Local or Atlas)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone [repository-url]
    cd wellnutrition
    ```

2.  **Setup Backend**:
    ```bash
    cd server
    npm install
    cp .env.example .env
    # Add your MONGODB_URI and JWT_SECRET to .env
    npm start
    ```

3.  **Setup Frontend**:
    ```bash
    cd client
    npm install
    npm run dev
    ```
